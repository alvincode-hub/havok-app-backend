Havok API Demo

Endpoints disponibles

Cette instance Render expose uniquement l'API. En mode demo, les routes de lecture sont accessibles directement.

Healthcheck: /api/health

GET /api/health

Verifie que le serveur repond.

GET /api/home

Retourne le payload principal d'accueil.

GET /api/tournaments/calendrier

Retourne le calendrier des tournois.

GET /api/tournaments/allWindow?eventId=... ou ?windowId=...

Retourne un groupe d'event via eventId ou windowId.

GET /api/tournaments/window?windowId=...

Retourne le detail d'une window.

GET /api/tournaments/results?windowId=...&page=0&cumulatif=false

Retourne les resultats pagines d'une window.

  • windowId requis
  • page optionnel
  • cumulatif=true pour le classement cumulatif
GET /api/players

Retourne la liste simplifiee des joueurs suivis.

GET /api/player?playerId=...

Retourne le profil complet d'un joueur.

POST /api/app/challenge

Endpoint d'auth mobile conserve pour compatibilite API.

POST /api/app/session

Echange challenge + attestation contre une session courte.
